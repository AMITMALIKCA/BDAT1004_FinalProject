BDAT1004 Final Project (A Dashboard for Crop Production in India)

Project Description: This is a dataset visualization webpage designed to analyze a crop production dataset consisting of approximately 248k entries, but we used 2000 sample batch.

The project was developed by a team of four members, namely Amit Malik, Khushi, Manpreet, and Viraj.

The dataset includes information on various crops produced in India, such as their production quantity, area of cultivation, and yield per hectare, among others. The webpage allows users to explore the dataset through different visualization tools and gain insights into the trends and patterns in crop production over the years.

The project utilizes open-source cloud services to store the dataset, which is publicly accessible through the link https://india-crop.s3.us-east-2.amazonaws.com/crop-dataset.csv. The code for the project is available on the Github repository at https://github.com/AMITMALIKCA/BDAT1004_FinalProject.
